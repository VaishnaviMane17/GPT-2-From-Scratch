# GPT-2-From-Scratch

This project implements GPT-2 from scratch using PyTorch, for understanding the mechanics of modern Large Language Models (LLMs).The goal of this implementation is to recreate and learn the core architecture manually, including token embeddings, transformer blocks, attention mechanisms, and autoregressive text generation.

By building GPT-2 step by step, this project demonstrates how modern language models learn contextual relationships between words and generate coherent text. Transformer models like GPT-2 rely on self-attention, feed-forward networks, residual connections, and positional embeddings to process sequential data effectively.
